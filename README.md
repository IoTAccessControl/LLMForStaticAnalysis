
## 依赖安装

1. 安装miniconda  
下载和安装：https://repo.anaconda.com/miniconda/  
创建python env：  
```
conda create -n agent-dev python=3.14
conda activate agent-dev 
```

2. 安装本项目的依赖
```
pip install -r requirements.txt
```
